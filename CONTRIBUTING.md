# Contributing

Thank you for your interest in improving this template.

## Rules
- This is a **template platform**, not an app.
- Do not add business logic, demo content, or branding.
- All changes must be reusable.

## Workflow
1. Fork the repository
2. Create a feature branch
3. Make changes
4. Run lint and build
5. Open a pull request

## Versioning
Breaking changes require a major version bump.

All releases are tagged using Semantic Versioning.
